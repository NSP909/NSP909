## Selected Projects

**[FocusFlow](https://devpost.com/software/focus-flow-y63tzm)** (1st, HackUMBC 2025). Real-time EEG attention classifier. Bidirectional LSTM with cross-hemispherical attention, 87% accuracy across 5 attention states. Async WebSocket pipeline handling 1,280 EEG samples/sec at sub-100ms latency, with a live Next.js dashboard.

**[TaxDaddy](https://devpost.com/software/taxdaddy)** (3rd, Hacklytics 2025 at Georgia Tech). Real-time voice tax agent over Twilio and WebSockets, coordinating 5 specialized agents through a multi-agent graph for live RAG and tool calling. OCR form parsing into MongoDB in under 7 seconds, plus Benford's-Law fraud detection.

**[SixthSense](https://devpost.com/software/sixthsense-xuw41r)** (Winner). Sensory substitution system that turns vision into touch for accessibility.

**[SolSpeak](https://devpost.com/software/n-kp7oqy)** (Solana/Blockchain Track Winner, HackNYU 2025). Voice-driven trading on Solana.

**Pulsify** (Honorable Mention, BioHack 2024). Healthcare query pipeline routing natural language through 3 parallel channels (records, trials, imaging) with NLP-to-SQL at 87% accuracy under 3s, plus multimodal RAG over a custom Llama 3.2 3B and LLaVA projection layer.

**L-LMS** (Fetch.ai "Beyond Boundaries" Track Winner, HackMIT 2024). Multi-agent academic support built on LangGraph with a 15,000+ embedding vector DB, Redis caching for sub-2s responses, and a hybrid Llama 3.1 / GPT-4o setup wired to the Canvas API.

Full list including HireFlow, Vector Mentor, AnchorVision, OpenSwarm, and Pitcher lives on [Devpost](https://devpost.com/nspriyadarshan).
